# Martial Arts Sim

Game is Live @ [Netlify](https://admiring-keller-da4bc7.netlify.app/) and @ [github]()

### Game Controls

- `Up Arrow` to move Forward
- `Down Arrow` to move Backward
- `Left Arrow` to Kick
- `Right Arrow` to Punch
- `Spacebar` to Block

* Press the `sound` button to toggle sound On or Off .
